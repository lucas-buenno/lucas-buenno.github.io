Projetos

Plataforma que detalha dados
de incidentes aéreos no Brasil

A Torre de Controle é um projeto criado através de dados públicos do gov.br que detalha acidentes aéreos no Brasil desde 2007. A plataforma mostra detalhes como, origem e destino do vôo, gravidade do acidente, quantidade de vítimas, detalhes das aeronaves, relatório do CENIPA com todas as causas do acidente.

Tecnologias usadas:

Java • Spring Boot • Postgres • Docker • Specifications • GraphQL
